# Assets Folder

## Images

Place your logo files in the `images/` folder:

### Required Files:
- **`rdt-logo.png`** - Your Recovery Dairon Tech logo (recommended: 1200x600px or square format)
  - Suggested dimensions: 600x400px for web optimization
  - Format: PNG with transparency recommended
  - File size: Keep under 100KB for optimal performance

### Logo Integration Points:
1. **Navigation Bar** - Top-left corner (40-48px height on desktop, 32px on mobile)
2. **Hero Section** - Center showcase below hero buttons (280px max width on desktop)
3. **Fallback** - Displays RDT Tech text if image fails to load

### Optimization Tips:
- Compress your PNG using tools like TinyPNG or ImageOptim
- Use WebP format for even better performance (add as fallback)
- Ensure logo has proper transparency/background for dark theme compatibility
- Test logo appearance on dark backgrounds
